# 🎁 Okazje.pl - Nowoczesna Platforma Okazji

Nowoczesna, responsywna strona internetowa do dzielenia się najlepszymi okazjami, promocjami i rabatami. Wzorowana na Pepper.pl z nowoczesnymi funkcjonościami.

## ✨ Funkcjonalności

### 📋 Strona Główna
- **Lista Okazji** - Wyświetlanie okazji w nowoczesnym formacie kartami
- **Filtrowanie** - Filtrowanie po kategoriach i statusie popularności
- **Sortowanie** - Sortowanie po najnowszych, najpopularniejszych, najgorętszych
- **Wyszukiwanie** - Szybkie wyszukiwanie okazji po tytule, opisie i sklepie

### 🔥 Wskaźnik Temperatury
- Wyświetla popularność okazji w skali od 0 do 100 stopni
- Kolory:
  - 🔴 Gorące (≥80°) - Najgorętsze oferty
  - 🟠 Ciepłe (60-79°) - Popularne
  - 🔵 Zimne (<60°) - Nowe lub mniej popularne

### 💰 Informacje o Cenie
- Cena Oryginalna
- Cena Promocyjna
- Procent rabatu
- Automatyczne obliczanie zniżek

### 👥 Interakcje Społeczne
- Licznik komentarzy
- Licznik zapisanych okazji
- Przycisk do zapisywania okazji
- Możliwość komentowania

### 🏪 Kategorie
- Elektronika
- Moda
- Dom i Ogród
- Sport i Turystyka
- Książki

### ➕ Dodawanie Okazji
- Modal do dodawania nowych okazji
- Formularz z walidacją
- Wybór kategorii
- Wprowadzanie cen

## 🛠️ Technologia

- **Fully Responsive** - Działa idealnie na mobilnych, tabletach i desktopach
